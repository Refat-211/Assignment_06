Ans: In JavaScript, users can declare a variable using 3 keywords that are var, let, and const. The scope of a var variable is functional scope. The scope of a let variable is block scope. The scope of a const variable is block scope. It can be updated and re-declared into the scope.

Ans: Unlike regular functions, arrow functions do not have their own this . The value of this inside an arrow function remains the same throughout the lifecycle of the function and is always bound to the value of this in the closest non-arrow parent function. A programmer can get the same result as regular functions by writing a few lines of code using arrow functions. Curly brackets are not required if only one expression is present. 'this' value inside a regular function is dynamic and depends on how the function is invoked. But 'this' inside the arrow function is bound lexically and equals to 'this' of the outer function. Fat arrow methods always bind 'this' a value to the class instance.
